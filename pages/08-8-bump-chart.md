## 학습 목표

- 범프 차트의 개념과 활용 목적을 이해합니다.
- 시간 흐름에 따른 순위 변화를 해석할 수 있습니다.
- Tableau에서 순위 계산과 선 마크를 활용해 범프 차트를 구현할 수 있습니다.

## 목차

1. 범프 차트란?
2. 범프 차트를 자주 쓰는 이유
3. Tableau에서 범프 차트 만드는 방법

## 1. 범프 차트란?

범프 차트는 시간의 흐름에 따라 항목의 순위 변화를 선의 위치 이동으로 표현하는 차트입니다.

- 절대값 자체보다 순위 변동을 강조합니다.
- 경쟁 구도와 순위 변화 흐름을 보여 주는 데 적합합니다.
- 특정 항목이 언제 순위가 오르거나 내려갔는지 직관적으로 읽을 수 있습니다.

즉, 범프 차트는 “값이 얼마나 큰가”보다 “순위가 어떻게 바뀌는가”를 보여주는 차트입니다.

## 2. 범프 차트를 자주 쓰는 이유

범프 차트는 여러 항목이 시간 흐름 속에서 서로 순위를 다투는 상황을 볼 때 강합니다.

대표적인 활용 예시는 다음과 같습니다.

- 기간별 브랜드 순위 변화
- 매출 상위 제품 순위 추이
- 콘텐츠 인기 순위 변동 분석

실무에서는 절대값 차트만 보면 다음을 놓치기 쉽습니다.

- 누가 새롭게 상위권에 진입했는가
- 누가 지속적으로 순위를 유지했는가
- 특정 시점에 순위가 급격히 바뀐 항목은 무엇인가

즉, 범프 차트는 `순위 경쟁의 스토리`를 읽는 데 적합합니다.

## 3. Tableau에서 범프 차트 만드는 방법

Tableau에서는 `RANK` 테이블 계산을 활용해 기간별 순위를 만든 뒤, 날짜를 축에 배치하고 선(Line) 마크로 연결하여 구현합니다.

기본 흐름은 다음과 같습니다.

1. 날짜 필드를 열 또는 행에 배치합니다.
2. 항목 차원을 세부 정보 또는 색상에 배치합니다.
3. 측정값을 기준으로 순위를 계산합니다.
4. `RANK()` 또는 `퀵 테이블 계산 -> 순위`를 적용합니다.
5. 마크 유형을 `선(Line)`으로 바꿉니다.
6. 축을 반전해 1위가 위쪽에 오도록 맞춥니다.

실무적으로 중요한 포인트는 다음과 같습니다.

- 축을 반전해야 상위 순위가 위에 보여 해석이 자연스럽습니다.
- 항목 수가 너무 많으면 읽기 어려워지므로 상위 N개만 필터링하는 경우가 많습니다.
- 강조할 대상만 진한 색, 나머지는 옅은 색으로 두면 가독성이 좋아집니다.

예를 들어:

- 열: 월
- 행: 순위 계산식
- 색상: 브랜드명
- 세부 정보: 브랜드명

처럼 구성하면 기간별 브랜드 순위 흐름을 선으로 연결해 볼 수 있습니다.

즉, 범프 차트의 핵심은 `시간 축 + 순위 계산 + 선 연결 + 축 반전`입니다.
