## 1. Tableau Agent란?

Tableau Agent는 Tableau 안에서 자연어로 시각화, 계산식, 데이터 탐색을 보조하는 생성형 AI 기능입니다.  
공식 도움말 기준으로 Tableau Desktop, Tableau Cloud, Tableau Server 웹 작성 환경에서 사용할 수 있으며, 버전과 환경에 따라 지원 범위가 달라집니다.

핵심은 “챗봇”이 아니라 `분석 작성 보조 도구`라는 점입니다.

## 2. Tableau Agent가 잘하는 일

공식 문서 기준으로 Tableau Agent는 다음 작업을 도와줄 수 있습니다.

- 시각화 만들기
- 적절한 차트 유형 제안
- 계산식 생성 및 설명
- 날짜 기반 분석
- 필터링과 정렬
- 데이터 탐색 질문 제안

즉, 빈 시트에서 무엇부터 시작해야 할지 막막할 때 초안을 빠르게 만드는 데 특히 유용합니다.

## 3. Tableau Agent의 한계

공식 FAQ에서는 다음과 같은 한계를 분명히 적고 있습니다.

- 데이터 원본 선택은 대신해 주지 못함
- 데이터 모델링은 해주지 못함
- 대시보드를 완성형으로 만들어 주지 못함
- 컨설팅 질문에 대한 정답을 주는 용도는 아님
- 필터 컨트롤, 매개변수 같은 인터랙션 설계는 제한적임

즉, Agent는 분석가를 완전히 대체하는 기능이 아니라, `초안 작성과 반복 작업을 줄여주는 보조 수단`으로 보는 편이 정확합니다.

## 4. 실무 활용 기준

공식 도움말에서 권장하는 좋은 사용 조건은 다음과 같습니다.

- 필드명이 명확하고 중복이 적을 것
- 차원/측정값 역할이 올바를 것
- 데이터 타입이 제대로 지정되어 있을 것
- 가능하면 추출 데이터나 정돈된 파일 데이터일 것

실무적으로는 다음 원칙이 중요합니다.

- Agent가 만든 계산식과 차트는 반드시 검토합니다.
- 질문은 한 번에 너무 많이 요구하지 않고 잘게 나눕니다.
- “무엇을 보고 싶은지”를 구체적으로 말할수록 결과가 좋아집니다.

예를 들어:

- 좋지 않은 요청: `분석 좀 해줘`
- 좋은 요청: `2024년 지역별 매출을 막대 차트로 보여주고, 매출 상위 5개만 남겨줘`

즉, Tableau Agent는 분석 자동화보다 `분석 가속화`에 더 가까운 도구입니다.
