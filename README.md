# 태블로를 활용한 데이터 시각화

`태블로를 활용한 데이터 시각화`는 Tableau를 처음 접하는 독자부터, 실무에서 대시보드와 시각화 품질을 높이고 싶은 사용자까지 폭넓게 고려한 입문서입니다.

이 책은 단순히 기능을 나열하는 방식이 아니라, 비즈니스 인텔리전스의 맥락 안에서 Tableau를 이해하고 실제 분석과 시각화에 연결하는 흐름으로 구성되어 있습니다. 따라서 "어떻게 클릭하는가"보다 "왜 이렇게 분석하고 시각화하는가"를 함께 익히는 데 초점을 둡니다.

## 이 책이 다루는 내용

- BI와 Tableau의 기본 개념
- Tableau Desktop 설치와 기본 조작
- 다양한 차트와 데이터 시각화 방법
- 계산된 필드, 계산 수준, 매개변수, 주요 함수
- 데이터 결합, 정리, 변형과 Tableau Prep
- 대시보드 설계, 구성, 공유와 Tableau Cloud

## 이런 독자를 위한 책입니다

- Tableau를 처음 배우는 입문자
- 엑셀 중심 보고에서 BI 도구 기반 분석으로 넘어가고 싶은 실무자
- 차트 제작을 넘어 해석 가능한 대시보드를 만들고 싶은 분석가
- Tableau 기능뿐 아니라 활용 맥락까지 이해하고 싶은 학습자

## 책의 구성

이 책은 총 5개의 챕터로 구성되어 있습니다.

1. Tableau 기초
2. 데이터 시각화
3. 데이터 계산 및 함수
4. 데이터 결합, 정리, 변형
5. 대시보드 구축 및 공유

각 챕터는 개념 이해, 기능 설명, 실습 흐름, 실무 활용 포인트가 자연스럽게 이어지도록 설계했습니다.
