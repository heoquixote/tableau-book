# 태블로를 활용한 데이터 시각화

차트를 만드는 방법부터 `왜 이 화면을 그렇게 만들어야 하는지`까지 다 담을 수 있게 제작했습니다.  
설치, 시각화, 계산, 전처리, 대시보드, Cloud 공유까지 실무 흐름대로 한 번에 익힐 수 있도록 구성했습니다.

## 이 저장소에서 다루는 핵심 주제

- Tableau와 BI 관점 이해
- Tableau UI와 기본 필드 개념
- 차트 작성과 분석 기능 활용
- 계산된 필드, LOD, 매개변수, 주요 함수
- 데이터 결합, 라이브 vs 추출, 데이터 원본 편집
- 대시보드 설계, 구성, 액션, 저장과 전달
- Tableau Prep 기반 전처리
- Tableau Server / Tableau Cloud를 통한 게시와 협업

## 이런 분께 맞습니다

- Tableau를 처음 배우는 분
- Excel 중심 보고에서 BI 도구 기반 분석으로 넘어가려는 분
- 차트 작성보다 “좋은 대시보드 설계”를 더 잘하고 싶은 분
- Tableau Cloud까지 포함해 공유와 운영 흐름을 익히고 싶은 분

## 도서 구성

### 01. Tableau 개념

BI가 왜 필요한지, 시각화가 왜 중요한지부터 시작합니다.  
이 장에서는 Tableau라는 도구를 단순히 소개하는 데서 끝나지 않고, 제품군 구조와 학습에 필요한 기본 맥락까지 큰 그림으로 정리합니다.

주요 내용:

- 시각적 분석(Visual Analytics)과 BI의 기본 개념
- 대표적인 BI 도구 비교와 Tableau의 위치
- Tableau 제품군, 구조, 라이선스 이해
- Tableau를 선택하는 이유와 강점
- Tableau 학습을 위한 기본 연결 흐름

### 02. Tableau 설치

실습 환경을 실제로 준비하는 장입니다.  
Tableau Desktop 설치부터 Tableau Cloud 계정 준비, 추가 인증, 라이선스 활성화와 확인, 실습 데이터 이해까지 이어지는 흐름을 다룹니다.

주요 내용:

- Tableau Desktop 다운로드와 설치
- 운영체제별 설치 흐름
- Tableau Cloud 계정 활성화
- MFA 등록과 인증 도구 설정
- Desktop 라이선스 활성화와 확인
- Tableau Desktop Public Edition과 유료 Desktop 차이
- Tableau Desktop Public Edition 설치
- 실습 데이터 소개

### 03. Tableau UI 및 기본 기능

Tableau를 처음 열었을 때 바로 마주치는 화면과 필드 개념을 정리하는 장입니다.  
데이터 패널, 분석 패널, 선반, 시트 영역이 각각 어떤 역할을 하는지, 그리고 차원과 측정값을 어떻게 구분해야 하는지와 정렬/필터, 기본 시각화 생성 방식까지 다룹니다.

주요 내용:

- Tableau UI의 주요 영역과 역할
- 데이터 패널과 선반에서 보이는 필드 아이콘
- 불연속형/연속형 시각적 단서
- 차원과 측정값의 차이
- 정렬과 필터의 기본 사용법
- Tableau에서 차트를 만드는 기본 방법

### 04. 기본 차트 시각화

Tableau에서 가장 많이 쓰는 기본 차트와 분석 기능을 익히는 장입니다.  
막대, 라인, 맵, 파이, 트리맵 같은 기본 차트부터 크로스탭, 스캐터 차트, 이중축, KPI 시각화까지 이어집니다.

주요 내용:

- 막대 차트, 라인 차트, 맵 차트, 파이 차트
- 트리맵, 버블 차트, 워드 클라우드
- 크로스탭, 테이블 계산, 히트맵
- 스캐터 차트, 측정값 집계, 분석 패널
- 이중축, 도넛 차트, 버터플라이 차트
- KPI 시각화

### 05. 데이터 필드 생성 및 활용

Tableau에서 자주 쓰는 기본 필드 구성을 다루는 장입니다.  
그룹, 집합, 구간 차원, 측정값 이름/측정값처럼 시각화 전에 데이터를 어떻게 다시 조직할지에 집중합니다.

주요 내용:

- 그룹을 활용한 범주 재분류
- 집합과 집합 기반 드릴다운
- 구간 차원과 히스토그램
- Measure Names / Measure Values 활용

### 06. 계산된 필드

계산식을 통해 분석을 정교하게 만드는 장입니다.  
계산된 필드, 계산 수준, 매개변수, 주요 함수까지 이어지는 계산 중심 흐름을 다룹니다.

주요 내용:

- 계산된 필드 작성과 계산식 구조 이해
- 행 수준 계산과 집계 수준 계산 비교
- 문자열, 숫자, 날짜, 순위 함수
- LOD 표현식
- 매개변수와 계산식을 활용한 동적 뷰 구성
- 전년 대비 KPI 계산

### 07. 데이터 결합, 정리, 변형

분석용 데이터 모델을 어떻게 연결하고 운영할지 다루는 장입니다.  
데이터 결합, 라이브와 추출 연결 차이, Tableau 작업 순서, 통합 문서 합치기와 데이터 원본 교체까지 이어집니다.

주요 내용:

- 관계, 조인, 유니온, 블렌딩 개념 비교
- 라이브 vs 추출 연결 방식 비교
- Tableau 작업 순서와 필터 적용 순서
- 통합 문서 합치기
- 데이터 원본 바꾸기와 참조 바꾸기

### 08. 다양한 차트 시각화

기초 차트 이후에 자주 쓰는 확장형 차트를 따로 모아 다루는 장입니다.  
간트 차트, 박스 플롯, 워터폴 차트, 불릿 차트, 퍼널 차트, 와플 차트, 파레토 차트, 범프 차트, 롤리팝 차트, 덤벨 차트, 생키 차트, 네트워크 차트, 레이더 차트, 썬버스트 차트처럼 목적이 분명한 차트들을 실무 관점으로 정리합니다.

주요 내용:

- 간트 차트
- 박스 플롯
- 워터폴 차트
- 불릿 차트
- 퍼널 차트
- 와플 차트
- 파레토 차트
- 범프 차트
- 롤리팝 차트
- 덤벨 차트
- 생키 차트
- 네트워크 차트
- 레이더 차트
- 썬버스트 차트

### 09. 대시보드 구축 및 공유

앞에서 만든 시트와 계산을 실제 대시보드로 묶고, 저장과 전달까지 연결하는 장입니다.  
대시보드의 역할과 목적을 이해한 뒤, 화면 설계와 디자인, Tableau Desktop에서의 레이아웃 구성, 액션과 Tooltip, 결과 저장과 내보내기까지 이어집니다.

주요 내용:

- 대시보드의 개념, 목적, 유형, 산업별 예시
- 비즈니스 질문 기반 화면 설계와 디자인 원칙
- 장치별 레이아웃, 대시보드 크기, 컨테이너와 개체
- 액션, Tooltip, Viz in Tooltip, Dynamic Zone Visibility
- 통합 문서 저장, 이미지/PDF/PowerPoint/Excel 내보내기
- Tableau Public 게시와 외부 공유

### 10. 태블로를 활용한 고급 분석

기본 시각화와 대시보드 이후에 다룰 확장 분석 기능과 Tableau의 최신 AI 기능을 모아두는 장입니다.  
Python 연동, 지표 중심 경험, 생성형 AI 보조, AI 에이전트 연계까지 실무 관점으로 정리합니다.

주요 내용:

- TabPy를 통한 Python 연동
- Tableau Pulse
- Tableau Agent
- Tableau MCP

### 11. Tableau Prep을 활용한 데이터 전처리

Tableau Prep으로 데이터를 정리하고 결합하는 흐름을 별도 장으로 분리했습니다.  
Prep 소개, 설치, 활성화, 유니온/조인/정리/출력까지 전처리 관점으로 묶어 다룹니다.

### 12. Tableau Server / Tableau Cloud

완성된 대시보드를 조직 안에서 운영하고 협업하는 흐름을 다루는 장입니다.
