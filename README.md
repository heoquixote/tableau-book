# 태블로를 활용한 데이터 시각화

`태블로를 활용한 데이터 시각화`는 Tableau를 처음 접하는 독자부터, 실무에서 더 설득력 있는 대시보드와 더 해석력 있는 시각화를 만들고 싶은 사용자까지 함께 고려한 입문서입니다.

이 책은 Tableau의 기능을 순서대로 나열하는 방식으로 구성되지 않았습니다. 대신 비즈니스 인텔리전스의 맥락 안에서 Tableau를 이해하고, 데이터를 어떻게 읽고 해석하며, 어떤 방식으로 시각화해야 더 나은 의사결정으로 이어지는지를 함께 다룹니다.

따라서 이 책의 중심에는 단순한 조작법이 아니라 분석의 흐름이 있습니다. 독자는 "어디를 클릭해야 하는가"를 넘어서 "왜 이 차트를 선택해야 하는가", "왜 이 계산이 필요한가", "왜 이 대시보드 구성이 더 효과적인가"를 함께 익히게 됩니다. 즉, Tableau를 도구로 배우는 데서 그치지 않고, 실무에서 바로 사용할 수 있는 분석과 시각화의 사고방식까지 익히는 것이 이 책의 목표입니다.

## 이 책이 다루는 내용

- BI와 Tableau의 기본 개념
- Tableau Desktop 설치와 기본 조작
- 다양한 차트와 데이터 시각화 방법
- 계산된 필드, 계산 수준, 매개변수, 주요 함수
- 데이터 결합, 정리, 변형과 Tableau Prep
- 대시보드 설계, 구성, 공유와 Tableau Cloud

각 장은 개념 이해에서 시작해 기능 설명, 실습 흐름, 실무 활용 포인트로 자연스럽게 이어지도록 구성했습니다. 그래서 처음 학습하는 독자도 따라올 수 있고, 이미 Tableau를 사용 중인 실무자도 자신의 분석 습관과 시각화 방식을 점검하며 읽을 수 있습니다.

## 이런 독자를 위한 책입니다

- Tableau를 처음 배우는 입문자
- 엑셀 중심 보고에서 BI 도구 기반 분석으로 넘어가고 싶은 실무자
- 차트 제작을 넘어 해석 가능한 대시보드를 만들고 싶은 분석가
- Tableau 기능뿐 아니라 활용 맥락까지 이해하고 싶은 학습자

특히 이 책은 "차트를 만드는 사람"보다 "데이터를 설명해야 하는 사람"에게 더 잘 맞습니다. 보고서를 작성해야 하는 실무자, KPI를 해석해야 하는 분석가, 여러 데이터를 하나의 화면에서 설득력 있게 보여줘야 하는 기획자와 운영 담당자에게 실질적인 도움이 되도록 설계했습니다.

## 책의 구성

이 책은 총 5개의 챕터로 구성되어 있습니다.

1. Tableau 기초
2. 데이터 시각화
3. 데이터 계산 및 함수
4. 데이터 결합, 정리, 변형
5. 대시보드 구축 및 공유

책의 흐름은 Tableau 기초에서 시작해 차트와 시각화, 계산과 함수, 데이터 준비, 대시보드 구축과 공유로 이어집니다. 즉, 단순히 기능 목록을 따라가는 것이 아니라 데이터를 연결하고, 계산하고, 시각화하고, 최종적으로 공유 가능한 결과물로 완성하는 전체 분석 흐름을 한 권 안에서 따라갈 수 있도록 구성했습니다.
